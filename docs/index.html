AI Playbook Policy Document Finder

Features

  • Finds and organizes authoritative public policy documents.
  • Supports manual seed URL mode.
  • Supports Google Custom Search API mode when credentials are available.
  • Generates CSV, Markdown, DOCX, and run log outputs.
  • Creates timestamped run folders.
  • Maintains run_index.csv.
  • Includes one-click batch launcher and tkinter GUI launcher.

How to Run

Run the quick demo from the project folder:

python ai_playbook_policy_document_finder.py --quick --max-results 3

Outputs

  • policy_document_candidates_RUNID.csv
  • policy_document_report_RUNID.md
  • policy_document_report_RUNID.docx
  • run_log_RUNID.txt

Responsible Use

This tool does not bypass logins, paywalls, or restricted content. Users must manually verify source authority, relevance, and currency.
